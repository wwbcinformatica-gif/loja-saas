## 🚀 **COMO USAR O PAINEL**

### **1. Acessar Painel:**
```
1. Fazer login como: wwbcinformatica@gmail.com
2. Ir para: /admin
3. ✅ Painel mestre carregado automaticamente
```

### **2. Criar Usuário:**
```
1. Clicar no botão "Novo Usuário" (verde)
2. Preencher dados de acesso (email, senha, nome da loja)
3. Preencher dados da loja (opcional, mas recomendado)
4. Selecionar plano e status
5. Clicar "Criar Usuário"
6. ✅ Usuário criado e disponível na lista
```

### **3. Editar Usuário:**
```
1. Na tabela, clicar "Editar" no usuário desejado
2. Modificar os campos necessários
3. Para alterar senha: preencher campo "Nova Senha"
4. Clicar "Salvar Alterações"
5. ✅ Dados atualizados na tabela
```

### **4. Deletar Usuário:**
```
1. Na tabela, clicar "Deletar" no usuário desejado
2. Confirmar lendo o aviso de segurança
3. Digitar "DELETAR" (em maiúsculas)
4. ✅ Usuário e todos os dados relacionados removidos
```

### **5. Buscar e Filtrar:**
```
1. Usar caixa de busca para encontrar por nome/email
2. Usar filtros de Status e Plano para refinar
3. ✅ Resultados atualizados em tempo real
```
